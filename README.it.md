# Archivio delle presentazioni in conferenze

Dal 2000 partecipo come relatore a conferenze tech italiane e internazionali, affrontando temi che attraversano l'intero panorama dello sviluppo software moderno: da Java, PHP e framework web fino a cluster Raspberry Pi, API dei social network, motori di ricerca open source, intelligenza artificiale e cloud computing.

Questo repository raccoglie tutti i talk in un unico punto. Ogni riga della tabella qui sotto rimanda a un repository GitHub dedicato, dove puoi trovare slide, esempi di codice e qualsiasi materiale aggiuntivo utilizzato durante la presentazione.

Che tu stia cercando ispirazione per la tua prossima proposta a una conferenza o voglia approfondire una tecnologia specifica, sentiti libero di esplorare, riusare e imparare.
